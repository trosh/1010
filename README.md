------------

HOW TO RUN
----------

    make run

    ./1010

CONTROLS
--------

- **1/2/3** TO CHOOSE BLOCK
- **ARROW KEYS** TO MOVE BLOCK
- **SPACE/RETURN** TO PLACE BLOCK
- **,/. or </>** TO SELECT PREVIOUS/NEXT BLOCK
- **ESCAPE/Q** TO QUIT
